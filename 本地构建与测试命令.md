# 本地项目构建与测试命令

## 构建测试（运行单元测试）
- App 单元测试：`./gradlew :app:testDebugUnitTest --warning-mode all`
- 全项目单元测试：`./gradlew :ai:test :common:test :document:test :search:test :highlight:test :tts:test --warning-mode all`

## 构建测试版本 APK
- 构建 Debug 测试版 APK：`./gradlew :app:assembleDebug -x test --warning-mode all`
